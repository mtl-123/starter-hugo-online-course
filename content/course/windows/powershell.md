# Windows 常用命令

[命令](https://www.hdd-tool.com/zh-TW/windows-11/disk-cleanup-on-c-drive.html)

## 一、清除 C 盘垃圾

```bat
1、按Windows + R 一起在键盘上
2、输入 " cleanmgr " 并按回车，默认选择C盘，直接点击OK以继续
```

## 二、手动清除所有垃圾

```bat
1、按Windows + R 一起在键盘上
2、输入 "  cleanmgr /LOWDISK " 然后按Enter
```

## 三、自动清理所有垃圾文件

```bat
1. 安装Windows + R 一起在键盘上
2. 输入 cleanmgr /VERYLOWDISK 然后按Enter
```
