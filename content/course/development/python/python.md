## 测试

{{< spoiler text="列表和元组之间的区别是什么?" >}}
Lists [列表]

- 列表是可变的——它们可以被更改
- 低于元组
- 语法: `a_list = [1, 2.0, 'Hello world']`

Tuples [元组]

- 元组是不可变的——它们不能被改变
- 元组比列表快
- 语法: `a_tuple = (1, 2.0, 'Hello world')`
