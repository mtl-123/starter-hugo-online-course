## 安装Git 
### Windows系统安装

Windows11 命令行安装：`winget install --id Git.Git -e --source winget`

[下载Git](https://github.com/git-for-windows/git/releases/download/v2.38.1.windows.1/Git-2.38.1-64-bit.exe)
### Linux系统安装

```bash
# Ubuntu 
apt-get install git -y
# CentOS
yum install git -y
```
